# Carpeta `mi-resultado/`

Aquí queda lo que **construiste con el kit**: el system prompt final de tu bot, capturas o transcripciones de pruebas, y una reflexión breve sobre lo que aprendiste.

Si estás siguiendo una formación que enlaza este kit, esta carpeta puede servirte como insumo del producto que entregas. Si lo usas por tu cuenta, te queda como bitácora documentada de tu primer asistente — útil para iterarlo después, para compartirlo con tu equipo, o para construir uno nuevo más adelante.

## Estructura sugerida

```
mi-resultado/
├── README.md                  # Este archivo
├── system-prompt-final.md     # El system prompt o configuración final de tu bot
├── reflexion.md               # Tu reflexión breve (200-400 palabras)
└── capturas/                  # Capturas o transcripciones de tus mejores pruebas
    └── [tus archivos aquí]
```

## Checklist genérica de un bot listo

### Tu system prompt final (`system-prompt-final.md`)
- [ ] Tiene los 4 componentes claramente etiquetados (ROL / ESTILO / CRITERIOS / RESTRICCIONES).
- [ ] Suma al menos 300 palabras.
- [ ] Restricciones incluyen al menos 3 reglas duras (sobre datos sensibles, derivación a humanos, no inventar).
- [ ] No tiene datos sensibles ni nombres reales de personas concretas.

### Tus pruebas (`capturas/`)
- [ ] Al menos 2 conversaciones: una típica y una de borde (fuera de rol).
- [ ] Datos personales anonimizados en todas las capturas.
- [ ] Cada captura está acompañada por una breve nota tuya sobre qué muestra y qué aprendiste.

### Tu reflexión (`reflexion.md`)
- [ ] Tiene entre 200 y 400 palabras.
- [ ] Articula: caso elegido + por qué te sirve + límites éticos detectados + qué ajustes hiciste.
- [ ] Identifica al menos un límite ético concreto del uso de IA en tu contexto.
- [ ] Termina con una proyección honesta: ¿lo vas a usar realmente? ¿En qué condiciones?

## Si esto es parte de una entrega formal

Si tu formación te exige entregar este resultado en alguna plataforma (LMS, Drive compartido, etc.), comprime la carpeta `mi-resultado/` en un ZIP y súbelo donde corresponda. Revisa los criterios de evaluación que aplique tu formación y asegúrate de cumplirlos antes de cerrar.

## Si solo es para ti

Quédate con la carpeta como bitácora. Cuando vuelvas a iterar el bot, parte desde aquí: vas a tener la versión final del prompt, qué probaste, y qué decidiste a propósito. Es un excelente punto de partida para la versión 2.
